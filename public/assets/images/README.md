# Carpeta de Imágenes - Terra Canada

## Instrucciones

Coloca aquí todas las imágenes del proyecto:

### Logo
- **Archivo**: `logo_terracanada.jpeg`
- **Ubicación**: `public/assets/images/logo_terracanada.jpeg`
- **Uso**: Login component
- **Dimensiones recomendadas**: 50x50px (se escala automáticamente)

### Otros Assets
Puedes organizar las imágenes por categoría:
- `public/assets/images/logos/` - Logos
- `public/assets/images/icons/` - Iconos
- `public/assets/images/backgrounds/` - Fondos
- `public/assets/images/illustrations/` - Ilustraciones

## Cómo Usar en Componentes

```html
<img src="assets/images/logo_terracanada.jpeg" alt="Terra Canada Logo" />
```

## Notas

- Las imágenes en la carpeta `public/` se sirven estáticamente
- Los cambios en esta carpeta requieren reiniciar el servidor de desarrollo
- Asegúrate de optimizar las imágenes antes de usarlas en producción
